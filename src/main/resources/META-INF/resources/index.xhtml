Agendamento de Vagas

Bem-vindo ao Sistema de Controle de Agendamentos de Vagas! Este projeto foi desenvolvido com o intuito de fornecer uma solução eficiente e escalável para o gerenciamento de agendamentos de vagas. O sistema permite o registro e consulta de vagas disponíveis, solicitantes e agendamentos, garantindo validações essenciais para manter a integridade dos dados. No back-end, utilizamos Java com Spring Boot, JPA/Hibernate e Lombok, enquanto no front-end utilizamos JSF com PrimeFaces, com o banco de dados HSQLDB.

As principais funcionalidades incluem o cadastro de vagas, onde é possível definir períodos e quantidade de vagas disponíveis; o cadastro de solicitantes, registrando suas informações; e o agendamento, com validações de disponibilidade e limites. Além disso, o sistema permite a visualização de agendamentos por período e a obtenção de estatísticas de uso.

Implementamos diversas validações, como a proibição de cadastro de vagas com datas retroativas, verificação de disponibilidade antes de novos agendamentos e controle da quantidade máxima de agendamentos permitida por solicitante. Explore as funcionalidades e utilize o sistema para gerenciar seus agendamentos de forma eficaz!
